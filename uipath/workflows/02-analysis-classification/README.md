# 02 Analysis / Classification

- Purpose/owner: analysis component creates factual findings, Raw Statement, and classification.
- Inputs: valid snapshot and user constraints.
- Outputs: analysis output and Raw Statement reference.
- Permitted: classify supported findings and identify missing context.
- Prohibited: fabricate evidence, mutate target, execute a correction.
- Entry: snapshot is available. Exit: findings trace to snapshot facts.
- Failure: retain facts and report unsupported or missing information.
- Receiver: 02.25 Recalibration.
- Authority: advisory only.
