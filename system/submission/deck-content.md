# Official presentation deck content

1. NextFlow: the deterministic pipeline for AI-generated configuration repairs.
2. Problem: generation is fast; proof and authority are the production bottleneck.
3. Solution: Maestro separates advisory reasoning, deterministic proof, and human authority.
4. Workflow: snapshot → agent analysis → readiness → simulation → replay → decision → apply → verify.
5. Safety contract: six admission conditions and exact ID/hash/target approval.
6. Demo case: `retry_limit=-1` becomes `3`; source and unrelated fields remain unchanged.
7. Failure handling: isolation, bounded retry, cancel/preserve, drift, application failure.
8. UiPath architecture: Maestro + Agent Builder + API Workflow + RPA + Action Center + storage.
9. Business adoption: reusable mappings let teams bind the same governed workflow to real resources.
10. Evidence and next step: live Automation Cloud run, package, public repository, and five-minute demo.
